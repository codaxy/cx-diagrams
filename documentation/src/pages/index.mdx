    # CxJS Diagrams

    This is a [CxJS](https://cxjs.io) library for building interactive diagrams such as flowcharts, network topologies, org charts, and data visualizations.

    **Capabilities:**
    - Draw shapes (rectangles, circles, rhombuses) with text, styling, and tooltips
    - Connect shapes with straight, L-shaped, or Z-shaped lines with arrow heads
    - Automatic layout using Flow containers for horizontal and vertical arrangements
    - Interactive diagrams with zoom, pan, click handlers, and context menus
    - Data-driven diagrams using CxJS data binding and repeaters
