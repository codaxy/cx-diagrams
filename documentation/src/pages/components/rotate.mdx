    # Rotate

    <ImportPath path='import { Rotate } from "cx-diagrams"' client:load />

    The `Rotate` component rotates its children by 90-degree increments. The `turns` property accepts values 0-3, where each increment represents a 90° clockwise rotation (0 = 0°, 1 = 90°, 2 = 180°, 3 = 270°). This is commonly used with network diagrams to lay out repetitive parts in different directions, often in combination with [`FourSides`](/components/four-sides).
