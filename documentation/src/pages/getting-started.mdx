    ## Key Concepts

    - **[Diagram](/components/diagram)** - The root container that provides zooming, panning, and the coordinate system
    - **[Cell](/components/cell)** - Defines a rectangular area that participates in layout flows
    - **[Shape](/components/shape)** - Visual elements like rectangles, circles, and rhombuses
    - **[Flow](/components/flow)** - Arranges cells in a directional layout (right, down, left, up)
    - **[Lines](/lines/straight-line)** - Connect shapes by referencing their `id` attributes
